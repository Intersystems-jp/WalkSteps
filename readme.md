# FlaskとEmbedded Pythonで歩数データをグラフ表示してみる

Flaskで簡単にWebアプリが作成できるので、Embedded Pythonでテーブル／グローバルから月毎の歩数データを取得し、matplotlibを利用してグラフを作成し、Flask Webアプリで表示させてみました。
